# Overview

This repository preserves the small, durable files and records that describe
the Windows gaming environment.

It is separate from:

- `BashScripts-WIN`, which contains general automation
- `dotfiles-WIN`, which contains development-environment configuration
- `E:/Video_Games`, which contains live installations and large assets

## Managed Areas

- Emulator configuration
- Per-title overrides
- Game and emulator metadata
- ReShade presets and installation records
- Recovery notes
- Directory and installation manifests

## Excluded Areas

- Game media
- Emulator binaries
- BIOS and firmware
- Virtual disks
- Saves and caches
- Generated archives
