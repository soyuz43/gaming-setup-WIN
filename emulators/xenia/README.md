# Xenia

Configuration and metadata for Xbox 360 emulation.

## Status

Planned. The emulator setup has not yet been added to the managed environment.

## Intended Repository Contents

### `configs/`

Preserved Xenia Canary global and per-title configuration files.

### `titles/`

Metadata for tested titles, patches, compatibility observations, and config
relationships.

## Excluded Assets

- Xbox 360 game media
- XEX files
- Emulator executables
- Content packages
- Title updates
- Save data
- Shader caches
- Download archives
