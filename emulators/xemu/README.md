# xemu

Configuration and metadata for the original Xbox emulator setup.

## Live Installation

- Root: `E:/Video_Games/Emulators/Xbox/xemu`
- Games: `E:/Video_Games/Emulators/Xbox/xemu/Games`
- Executable: `xemu.exe`
- Configuration location: to be confirmed

## Repository Directories

### `configs/`

Preserved global or per-title TOML configuration files.

### `titles/`

JSON metadata describing tested titles, source filenames, hashes, status,
configuration relationships, and troubleshooting notes.

## Excluded Assets

- ISO and XISO files
- Xbox BIOS
- MCPX ROM
- Xbox HDD image
- xemu executable
- Runtime logs and caches
