# Emulators

Emulator-specific configuration, title metadata, and recovery documentation live here.

Each managed emulator may contain:

- `README.md` — installation and recovery notes
- `configs/` — intentional global or per-title configuration
- `titles/` — structured metadata for individual titles

This directory must not contain emulator binaries, firmware, game media, virtual disks, saves, caches, or downloaded archives.
