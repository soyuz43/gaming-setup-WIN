# ReShade

Preserved ReShade presets and records describing where they are used.

## `presets/`

Contains intentionally maintained preset files.

## Installation Records

`manifests/reshade.tsv` records the relationship between a game, executable,
rendering API, preset, installation status, and notes.

## Excluded Files

- ReShade DLLs
- Game executables
- Installer packages
- Downloaded shader collections
- Screenshots and recordings
